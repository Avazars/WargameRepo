> [!Warning] Restrictions
> - Can only target an enemy unit that it can move into Base Contact by moving up to a number of inches using the first part of its movement stat.

> [!info] Step-Up & Strike
> When a unit performs a Step-Up & Strike [[Standard Action]]: 
> - That unit targets an enemy unit and moves any number of inches up to its first part of its [[Movement Stat]] to get into Base Contact with the target unit.   
> - This unit and the target unit become [[Engaged]] and both units gain the [[Engaged|Engaged Status]].
> - This unit performs a [[Strike]] against the targeted unit.

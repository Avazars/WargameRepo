## **Detachments**

Notes:
- Leader Units will determine at least one of the detachments that you have to take.
- Detachments will have special abilities tied to discarding the directives that they have.
- Detachments will be unique to each army.


Current list of Generic Detachment Types

| Detachment Type        | Directive Flavor                             |
| ---------------------- | -------------------------------------------- |
| [[Strike Detachment]]  | Scores by destroying enemy units.            |
| [[Hold Detachment]]    | Scores by maintaining control of objectives. |
| [[Capture Detachment]] | Scores by taking objectives from enemies.    |
| [[Claim Detachment]]   | Scores by board control.                     |
| [[Delve Detachment]]   | Scores by performing actions in terrain.     |
